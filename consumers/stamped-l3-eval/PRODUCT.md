# PRODUCT.md — Stamped L3 Eval Lab

## Register

`product`

## Product purpose

Internal forensic lab for Stamped engineers to see everything L3 engines, rulepacks, and ML shadow paths produce or suppress for a telemetry window.

## Users

- L3/ML engineers debugging false positives and misses
- Platform engineers verifying golden corpus / rulepack pins
- Not plant operators, CS, or customers (those use L6)

## Jobs to be done

1. Open a corpus window and see every detection status
2. Filter by category, rulepack URI, engine, or ML model
3. Compare two runs
4. Optionally attach to a running core lab export

## Tone

Quiet, dense, trustworthy laboratory tool. Task over brand theatre.

## Anti-references

- Customer plant dashboard (L6)
- Marketing landing aesthetics
- Purple glow “AI tool” dark mode clichés
- KPI stat strips on the first view

## Strategic principles

- Truth comes only from RunArtifact / core lab export
- Emitted and suppressed are peers in the UI
- Offline default; live is attach, not the only mode
